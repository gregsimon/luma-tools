## 2. Description of Service

Luma Tools is a client-side web application that provides the following functionality:

- **Sample Editing:** Convert, edit, and process audio samples for use with Luma-1 and Luma-Mu devices
- **Bank Management:** Assemble, organize, import, and export sample banks
- **Device Communication:** Read and write samples to/from Luma-1 and Luma-Mu devices via MIDI
- **Cloud Storage:** Optional integration with Google Drive for storing and sharing sound files
- **Pattern Editing:** Edit and manage pattern data for drum machines
- **MIDI Monitoring:** Monitor MIDI communication with connected devices

The Service operates entirely in your web browser and does not require server-side processing for core functionality.

## 3. System Requirements

Luma Tools requires:

- A modern web browser that supports WebMIDI (Chrome or Firefox recommended; Safari does not support WebMIDI)
- A MIDI interface or USB connection to communicate with Luma-1 or Luma-Mu devices (for device communication features)
- An active internet connection (for initial loading and optional Google Drive features)
